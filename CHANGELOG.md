# Changelog

All notable changes to the Obsidian Skill Viewer plugin are documented here.

## [1.0.0] — 2026-03-21

### Added
- `.skill` file registration — Obsidian opens `.skill` files natively
- **Skill View** — renders the ZIP archive as a rich note: header card, description, version/author/tags badges, embedded file tree, and full SKILL.md body rendered as Markdown
- **Skill Explorer** — sidebar panel listing all `.skill` files in the vault with search filter and one-click open
- **Settings tab** — toggle auto-open explorer on startup, adjust display options
- Gradient header card with skill metadata (name, version, author, tags)
- File tree showing all files inside the `.skill` ZIP archive
- Support for skills with or without YAML frontmatter in SKILL.md
